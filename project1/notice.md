## About this Project

This project is about food recipes, the main concept comes from the mobile game **"Food Fantasy"** which is a restaurant management RPG game.

## Feature

### Day-Night Cycle themes

The background color changes over time.

### Audio Player

This website has worked background music player, it can be mute, skip, pause.
